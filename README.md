🎙️ Projeto - Conversa por Voz com IA

Descrição
Este projeto foi desenvolvido como parte de um desafio da DIO.

A proposta é criar um sistema que utiliza voz para interagir com inteligência artificial, envolvendo:
- Speech-to-Text (voz para texto)
- Processamento com IA
- Text-to-Speech (texto para voz)

Tecnologias
- Python
- gTTS
- Conceitos de OpenAI (ChatGPT e Whisper)

Implementação atual
Nesta versão, foi implementada a conversão de texto em áudio utilizando gTTS.

Melhorias futuras
- Implementar reconhecimento de voz com Whisper
- Integrar com API do ChatGPT
- Criar fluxo completo de conversa por voz

Como executar
```bash
pip install -r requirements.txt
python main.py
